# 格式规范 - 写作标准与去AI痕迹指南

## 1. 中英文/数字混排规范

### 1.1 基本原则

**中文语境下，中英文、数字与中文之间不加空格。**

### 1.2 具体规则

| 场景 | ❌ 错误 | ✅ 正确 |
|------|--------|--------|
| 英文缩写 | MBA 论文 | MBA论文 |
| 英文单词 | AI 写作 | AI写作 |
| 数字+中文 | 15 % 的增长 | 15%的增长 |
| 数字+量词 | 5 个企业 | 5个企业 |
| 年份 | 2023 年 | 2023年 |
| 百分比 | 提升了 20 % | 提升了20% |
| 货币 | 100 万元 | 100万元 |
| 英文人名 | Smith (2019) | Smith(2019) |
| 英文标题 | IPD 理论 | IPD理论 |
| 版本号 | 第 3 代 | 第3代 |
| 编号 | 图 1-1 | 图1-1 |
| 章节 | 第 1 章 | 第1章 |

### 1.3 例外情况

以下情况保留空格：
- 英文句子中的中文（如：This is a 中文 sentence）
- 代码中的注释（如：# 这是一个 comment）
- 特殊格式要求（如期刊投稿要求）

### 1.4 检测方法

```python
# 检测中英文/数字间多余空格
import re

# 中文后接英文/数字有空格
pattern1 = r'[\u4e00-\u9fff]\s+[a-zA-Z0-9]'
# 英文/数字后接中文有空格  
pattern2 = r'[a-zA-Z0-9]\s+[\u4e00-\u9fff]'

# 合并检测
pattern = r'[\u4e00-\u9fff]\s+[a-zA-Z0-9]|[a-zA-Z0-9]\s+[\u4e00-\u9fff]'
```

## 2. 图表规范

### 2.1 图表基本要求

1. **自明性**: 图表应有自明性，不阅读正文也能理解
2. **清晰可读**: 图表清晰，不可直接拷贝模糊图片
3. **编号规范**: 图表必须编号，格式为"图1-1"、"表1-1"
4. **标题完整**: 图表应有图题/表题，说明图表内容

### 2.2 图的规范

#### 编号与标题
- 图序按章编排，如第1章第一个图为"图1-1"
- 图题置于图下方，居中
- 图题格式：图号+空格+图名（无标点）
- 示例：**图1-1 技术路线图**

#### 文字说明
- 图之前必须有引用提示，如"如图1-1所示"
- 图之后必须有文字说明和解释
- 先文字后图表，图表是文字的补充

#### 字体字号
- 图中文字：宋体/Times New Roman，5号或小5号
- 图题：宋体，5号，居中
- 同一图中字号统一

### 2.3 表的规范

#### 编号与标题
- 表序按章编排，如第1章第一个表为"表1-1"
- 表题置于表上方，居中
- 表题格式：表号+空格+表名（无标点）
- 示例：**表1-1 调研样本分布**

#### 格式要求
- 采用三线表格式（顶线、栏目线、底线）
- 表头设计简单明了，尽量不用斜线
- 表中文字：宋体/Times New Roman，5号或小5号
- 表内文字1倍行距

#### 数据规范
- 数据应准确无误，书写清楚
- 数字空缺的格内加横线"—"
- 同一指标小数位数保持一致
- 单位标注在表头右上角加圆括号

### 2.4 图表摆放

1. **图文配合**: 图表应与文字紧密配合，文图相符
2. **位置合理**: 图表尽量跟随首次提及的段落
3. **分页处理**: 图过大时可移至次页，图题只出现在下页
4. **留白规范**: 图表前后各空1行

## 3. 引用规范

### 3.1 正文引用

#### 引用格式
- 引用用上标序号，如"...成果[1]"
- 序号用小4号Times New Roman，置于方括号中
- 引用标注在引文最后一个句号之前
- 多次引用同一文献用同一序号

#### 多处引用
- 引用多篇：[1, 2, 5]
- 连续引用：[1-3]
- 注意：按增序排列

#### 直接说明
- 格式："由文献[8, 10-14]可知："
- 此时序号与正文排齐，不用上标

### 3.2 脚注

#### 使用场景
- 解释性说明
- 补充性信息
- 引用来源（网站、报纸等不宜入参考文献的资料）

#### 规范要求
- 脚注序号用①②③
- 每页重新编号
- 脚注内容小5号宋体
- **注意**：不引用百度百科、维基百科等网络百科

### 3.3 参考文献

#### 数量要求
- 参考文献不少于50篇
- 外文文献≥1/3
- 近5年文献≥1/3
- 近2年文献应有涉及

#### 类型要求
- 期刊论文≥60%
- 专著≤20%
- 避免大量引用学位论文
- 不引用百度百科、网页资料（除非是官方数据源）

#### 常用期刊
- 《管理世界》
- 《中国工业经济》
- 《南开管理评论》
- 《会计研究》
- 《金融研究》
- 《经济管理》
- 《财经研究》
- 《管理学报》

#### 著录格式

**期刊**:
```
[序号] 作者. 题名[J]. 期刊名, 年, 卷(期): 页码.
[1] 王学通, 崔雪竹. 大型公共项目交易模式溢出效应研究[J]. 建筑经济, 2014(8): 122-125.
[2] Helfat C E, Raubitschek R S. Product Sequencing[J]. Strategic Management Journal, 2000(21): 961-979.
```

**专著**:
```
[序号] 作者. 题名[M]. 出版地: 出版者, 出版年: 页码.
[3] 靖爽, 陈年根. 研发体系改进之道[M]. 天津: 天津人民出版社, 2019: 45-48.
```

**学位论文**:
```
[序号] 作者. 题名[D]. 保存地: 保存单位, 年份: 页码.
[4] 赖文勇. 基于过程视角的建筑企业社会责任行为识别与优化研究[D]. 广州: 广州大学, 2019: 45-48.
```

## 4. 数字与单位规范

### 4.1 数字用法

1. **阿拉伯数字**: 统计数据、年代、百分比、金额等
2. **汉字数字**: 概数(三五个)、成语(一帆风顺)、固定搭配(星期四)
3. **年份完整**: 2023年，不写23年
4. **百分比**: 统一使用%，如"提升了20%"

### 4.2 单位规范

1. **法定单位**: 使用国家法定计量单位
2. **单位符号**: 除人名命名的单位首字母大写外，一律小写
3. **中文单位**: 可用汉字与单位符号混写，如"万t·km"
4. **时刻表达**: 用中文计量单位，如"上午8点3刻"

## 5. 标点符号规范

### 5.1 常用标点

1. **句号**: 中文用"。"，英文用"."
2. **逗号**: 中文用"，"，英文用","
3. **分号**: 并列分句间用分号
4. **冒号**: 提示下文或总结上文
5. **引号**: 引用、强调、特殊含义用引号
6. **书名号**: 书名、篇名、报纸名用书名号《》

### 5.2 特殊用法

1. **破折号**: 减少使用"——"，可用逗号或括号替代
2. **省略号**: 用"……"(中文占两格)，不用"..."
3. **连接号**: 年份、页码范围用"-"，如"2021-2023年"、"45-48"

### 5.3 标点与空格

- 标点后不加空格(英文写作除外)
- 括号内文字与括号间不加空格
- 书名号之间不加顿号

## 6. 段落与层级规范

### 6.1 章节编号

```
第1章 绪论              (章)
1.1 研究背景            (节)
1.1.1 行业背景          (条)
1.1.1.1 细分市场        (款) - 一般不用
```

### 6.2 层级要求

1. **每章至少4节**（含本章小结）
2. **节下有目**: 避免节下直接到正文，应有3级标题
3. **内容充实**: 每节内容不少于1页

### 6.3 本章小结

- 每章最后应有"本章小结"
- 字数：200-600字
- 内容：回顾本章主要内容，承上启下
