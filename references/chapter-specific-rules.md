# MBA 论文章节特定规则

本文档详细说明 MBA 论文不同章节的写作规范和去 AI 痕迹策略。

## 一、文献综述章节

### 1.1 常见 AI 特征

#### 罗列式综述
- **特征**: 简单罗列作者和观点，缺乏深度分析
- **AI 示例**: "Smith(2010)认为...，Johnson(2012)指出...，Lee(2015)强调..."
- **改写策略**: 按主题或流派组织，比较不同观点的异同

#### 缺乏批判性
- **特征**: 只陈述不评价，没有指出研究局限
- **AI 示例**: "大量研究表明..."
- **改写策略**: 指出各研究的方法局限、样本问题或结论差异

#### 时间线堆砌
- **特征**: 按时间顺序简单罗列，缺乏逻辑主线
- **改写策略**: 按理论发展脉络或研究主题分类组织

### 1.2 改写原则

1. **主题化组织**: 不按作者罗列，按研究主题分类
2. **比较分析**: 对比不同研究的观点差异
3. **指出空白**: 明确现有研究的不足和本研究的切入点
4. **逻辑递进**: 从宏观到微观，或从理论到实证

### 1.3 规范示例

**❌ AI 风格**:
```
关于企业创新，Smith(2010)认为创新是企业发展的动力。Johnson(2012)指出创新需要组织支持。
Lee(2015)强调了领导力的作用。Wang(2018)研究了创新绩效的影响因素。
```

**✅ 改写后**:
```
企业创新研究主要从三个视角展开：资源基础观强调内部资源配置(Smith, 2010)，
组织支持理论关注制度环境(Johnson, 2012)，领导行为视角则聚焦高管特质(Lee, 2015)。
然而，现有研究多关注创新投入与产出的直接关系，对创新过程的动态机制探讨不足(Wang, 2018)。
本研究试图填补这一空白，从...视角分析...
```

## 二、案例分析章节

### 2.1 常见 AI 特征

#### 描述过多分析不足
- **特征**: 大段描述企业背景，缺乏深入分析
- **AI 示例**: "该公司成立于1990年，总部位于北京，员工5000人，年营收10亿元..."
- **改写策略**: 精简背景，聚焦与研究问题相关的关键信息

#### 结论先行
- **特征**: 先下结论再找证据
- **AI 示例**: "由此可见，该公司战略非常成功。具体表现在..."
- **改写策略**: 先呈现事实和数据，再推导结论

#### 缺乏数据支撑
- **特征**: 使用"显著"、"大幅"等模糊词汇，无具体数据
- **改写策略**: 提供具体数字、百分比、时间跨度

#### 模板化结构
- **特征**: 严格按照"背景-问题-措施-效果"四段式
- **改写策略**: 根据案例特点灵活组织，突出关键转折点

### 2.2 改写原则

1. **问题导向**: 围绕研究问题筛选信息，删除无关背景
2. **数据说话**: 用具体数字替代模糊描述
3. **过程追踪**: 关注决策过程和因果关系
4. **批判视角**: 不仅说成功，也分析失败或局限

### 2.3 规范示例

**❌ AI 风格**:
```
华为公司是全球领先的通信设备制造商。公司成立于1987年，总部位于深圳。
经过30多年发展，已成为世界500强企业。华为非常重视研发创新，每年投入大量资金。
这种战略使华为在5G领域取得领先地位。
```

**✅ 改写后**:
```
华为的研发投入策略值得深入研究。2012-2022年间，华为累计研发投入超过9770亿元，
占年营收比例从13%提升至22%(华为年报，2022)。这一战略转变源于2012年任正非
"备胎计划"的内部讲话，要求关键零部件自主可控。2019年美国制裁后，海思芯片
从"备胎"转正，2020年华为手机业务中采用自研芯片的比例从15%提升至65%(IDC, 2021)。
然而，这一战略也带来挑战：2021年消费者业务收入同比下降49.6%，
迫使华为加速拓展企业业务和云计算(华为年报，2021)。
```

## 三、战略建议章节

### 3.1 常见 AI 特征

#### 空泛建议
- **特征**: 建议过于笼统，缺乏可操作性
- **AI 示例**: "企业应加强创新、优化管理、提升服务"
- **改写策略**: 具体说明做什么、怎么做、谁来做

#### 忽视约束条件
- **特征**: 不考虑企业资源限制和外部环境
- **AI 示例**: "建议企业投入巨资进行数字化转型"
- **改写策略**: 分析实施条件、资源需求、潜在风险

#### 套用模板
- **特征**: SWOT、PEST等分析流于形式
- **改写策略**: 根据企业实际情况选择分析框架，深入挖掘关键因素

#### 缺乏优先级
- **特征**: 建议不分轻重缓急
- **改写策略**: 按紧急性和重要性排序，明确实施阶段

### 3.2 改写原则

1. **具体可操作**: 明确行动主体、具体措施、预期效果
2. **考虑约束**: 分析资源、能力、环境的限制
3. **分阶段实施**: 区分短期、中期、长期建议
4. **风险评估**: 指出实施建议的潜在风险和应对

### 3.3 规范示例

**❌ AI 风格**:
```
基于以上分析，建议该公司：第一，加强数字化转型；第二，优化组织架构；
第三，提升员工能力；第四，拓展新市场。通过这些措施，企业将实现可持续发展。
```

**✅ 改写后**:
```
针对该公司现状，提出以下建议：

短期（6个月内）：鉴于现金流紧张，建议优先实施低成本数字化措施。
具体包括：将客户管理系统迁移至云端（预计投入5万元，节省IT维护成本30%），
使用开源数据分析工具替代现有商业软件（年节省许可费12万元）。
实施主体为IT部门，需招聘1名云架构师（月薪1.5万元）。

中期（1-2年）：在现金流改善后，推进组织扁平化改革。
当前管理层级为5层，决策周期平均14天。建议压缩至3层，授权区域经理
直接审批50万元以下项目。预计可减少管理岗位8个，年节省人力成本120万元，
决策周期缩短至5天。风险在于中层管理人员抵触，需配套转岗培训方案。

长期（3-5年）：基于核心能力评估，建议聚焦华东地区市场，暂缓全国扩张。
理由：公司在华东品牌认知度达65%，而华南仅12%，进入新市场需投入3000万元
以上，且面临已有强势竞争者。集中资源深耕华东，目标3年内市场份额从18%提升至25%。
```

## 四、章节通用检查清单

### 4.1 文献综述
- [ ] 按主题组织而非按作者罗列
- [ ] 有比较分析而非简单陈述
- [ ] 指出研究空白和本研究定位
- [ ] 引用格式规范统一

### 4.2 案例分析
- [ ] 围绕研究问题筛选信息
- [ ] 提供具体数据和时间节点
- [ ] 分析因果机制而非描述现象
- [ ] 有批判性视角

### 4.3 战略建议
- [ ] 建议具体可操作
- [ ] 考虑资源和能力约束
- [ ] 明确实施阶段和优先级
- [ ] 评估潜在风险

## 五、中英文/数字混排规范

### 5.1 基本原则

**中文语境下，中英文、数字与中文之间不加空格。**

### 5.2 具体规则

| 场景 | ❌ 错误 | ✅ 正确 |
|------|--------|--------|
| 英文缩写 | MBA 论文 | MBA论文 |
| 英文单词 | AI 写作 | AI写作 |
| 数字+中文 | 15 % 的增长 | 15%的增长 |
| 数字+量词 | 5 个企业 | 5个企业 |
| 年份 | 2023 年 | 2023年 |
| 百分比 | 提升了 20 % | 提升了20% |
| 货币 | 100 万元 | 100万元 |
| 英文人名 | Smith (2019) | Smith(2019) |

### 5.3 例外情况

以下情况保留空格：
- 英文句子中的中文（如：This is a 中文 sentence）
- 代码中的注释（如：# 这是一个 comment）
- 特殊格式要求（如期刊投稿要求）

### 5.4 检测正则表达式

```python
# 检测中英文/数字间多余空格
import re

# 中文后接英文/数字有空格
pattern1 = r'[\u4e00-\u9fff]\s+[a-zA-Z0-9]'
# 英文/数字后接中文有空格  
pattern2 = r'[a-zA-Z0-9]\s+[\u4e00-\u9fff]'

# 合并检测
pattern = r'[\u4e00-\u9fff]\s+[a-zA-Z0-9]|[a-zA-Z0-9]\s+[\u4e00-\u9fff]'
```

## 六、快速改写流程

1. **识别章节类型**: 确定是文献综述、案例分析还是战略建议
2. **应用对应规则**: 按本节规则检查改写
3. **混排格式检查**: 确保中英文/数字间无多余空格
4. **对照检查清单**: 使用对应章节的检查清单验证
5. **通读润色**: 确保语言自然流畅
